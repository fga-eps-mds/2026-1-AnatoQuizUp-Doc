## Critérios definidos para avaliação

### US - Loja Virtual

| Critério |
|----------|
| Acesso e Interface: a loja deve estar acessível pelo menu principal da plataforma. |
| Catálogo de itens: listar itens organizados por categoria (ícones de perfil, molduras, títulos, planos de fundo, avatares estáticos). |
| Informações dos Itens: cada item deve exibir nome, categoria, preço em moedas e imagem de pré-visualização. |
| Validação de Compra: verificar se o saldo do aluno é suficiente antes de concluir a compra. |
| Compra Bem-sucedida: descontar moedas, exibir mensagem de sucesso, registrar item e disponibilizá-lo na tela de personalização. |
| Saldo Insuficiente: bloquear a compra, não descontar moedas e informar o aluno. |
| Itens já Adquiridos: impedir recompra e identificar o item como adquirido na loja. |

### US - Personalizar Perfil

| Critério |
|----------|
| Acesso: tela de personalização acessível a partir da página de perfil. |
| Itens Disponíveis: permitir selecionar apenas itens pertencentes ao aluno por categoria. |
| Seleção Exclusiva: apenas um item selecionado por categoria por vez. |
| Persistência: escolhas permanecem salvas após encerrar e reiniciar a sessão. |
| Exibição no Perfil: ícone, moldura, título, plano de fundo e avatar refletem as escolhas salvas. |
| Exibição em Outras Áreas: ícone, moldura e título exibidos na página de amizades. |

### US - Sistema de Conquistas e Marcos (Badges e Troféus)

| Critério |
|----------|
| Gatilhos de Conquista: disparar conquistas automaticamente por marcos de quantidade, especialidade e sequências (streaks). |
| Tipos de Recompensa: conquistas concedem medalhas/badges e moedas. |
| Notificação de Desbloqueio: exibir pop-up animado e sonoro no momento do desbloqueio. |
| Galeria de Conquistas: aba "Minhas Conquistas" com medalhas obtidas e bloqueadas. |
| Integração com o Perfil Público: aluno pode destacar até 3 conquistas no perfil. |
