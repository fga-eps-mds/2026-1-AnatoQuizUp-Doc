Eixo Produto - modelo de qualidade por repositorio

verde >= 0,75 (aprovado) amarelo 0,50-0,74 (atencao) vermelho < 0,50 (critico)

Como ler este painel

  • Qualidade do Produto = media ponderada das densidades por arquivo: Complexidade 35%, Comentarios 10%, Duplicacao 25% e Cobertura 30%.
  • Bloqueios = issues com descricao preenchida / total de issues do repositorio, quando houver export disponivel.
  • Prontidao = builds com sucesso / builds concluidas do repositorio.
  • O arquivo gerar-dados-produto.mjs escolhe sempre o JSON mais recente de cada repositorio em analytics-raw-data e regenera dados-produto.js.
